LVGL 9 XML Attributes - Quick Reference

Essential attributes for rapid UI development | LVGL 9.3 | Source: lvgl/src/others/xml/parsers/*.c

Layout & Position

Base Attributes

name str for lv_obj_find_by_name()
width size
height size
x size from left edge
y size from top edge
align enum position within parent
center, top_left, top_mid, top_right, bottom_*, left_mid, right_mid
Size values: px (100), % (50%), "content" (auto-size to content)

Flex Layout

Attributes

flex_flow enum direction of items
flex_grow int 0=fixed, 1+=fills space
flex_in_new_track bool wrap to next line
row, column, row_reverse, column_reverse, *_wrap

Style Properties

style_flex_main_place along flow axis
style_flex_cross_place perpendicular axis
start, center, end, space_between, space_around, space_evenly
⚠️ flex_align doesn't exist! Use style_flex_* instead

Grid Layout

style_grid_column_align
style_grid_row_align
style_grid_cell_column_pos col index
style_grid_cell_column_span cols wide
style_grid_cell_row_pos row index
style_grid_cell_row_span rows tall
align: start, end, center, stretch, space_*

lv_label

text string
long_mode enum text overflow
bind_text subject [fmt] reactive binding
long_mode: wrap, scroll, scroll_circular, dots, clip
bind_text: "subject_name '%.1f°C'"

lv_button / lv_checkbox

lv_button

Inherits all lv_obj attributes only

lv_checkbox

text string

lv_slider / lv_bar

value int [anim] current val
start_value int [anim] range mode
range min max "0 100"
bind_value subject reactive
mode enum
orientation enum
mode: normal, range, symmetrical | orientation: auto, horizontal, vertical

lv_image

src image
scale_x 256=100% 512=200%
scale_y 256=100% 128=50%
rotation 0.1° units 900=90°
inner_align enum within bounds
pivot x y rotation center
inner_align: center, stretch, tile, top_*, bottom_*
⚠️ zoom doesn't exist! Use scale_x/scale_y

Style: Size & Spacing

Padding

style_pad_all inside widget
style_pad_hor left+right
style_pad_ver top+bottom
style_pad_gap between flex items
style_pad_left/right
style_pad_top/bottom

Margin

style_margin_all outside widget
style_margin_hor
style_margin_ver

Size

style_radius rounded corners
style_width/height

Style: Background

style_bg_color 0xRRGGBB
style_bg_opa 0-255/% opacity
style_bg_grad_dir gradient direction
style_bg_grad_color end color
style_bg_image_src
style_bg_image_tiled repeat pattern
style_bg_image_recolor tint color
style_bg_image_recolor_opa tint strength
grad_dir: none, hor, ver

Style: Text

style_text_color
style_text_font font name
style_text_opa
style_text_align within label
style_text_letter_space kerning
style_text_line_space leading
style_text_decor
align: left, right, center, auto | decor: none, underline, strikethrough

Style: Border & Shadow

Border

style_border_color
style_border_width
style_border_opa
style_border_side which edges
side: none, top, bottom, left, right, full

Shadow

style_shadow_width blur amount
style_shadow_color
style_shadow_opa
style_shadow_offset_x horizontal shift
style_shadow_offset_y vertical shift

Style: Effects

style_opa 0-255 transparency
style_blend_mode compositing
style_transform_scale_x 256=100%
style_transform_scale_y 256=100%
style_transform_rotation 0.1° units
style_translate_x shift left/right
style_translate_y shift up/down
style_image_recolor tint color
style_image_recolor_opa tint strength
blend_mode: normal, additive, subtractive, multiply, difference
⚠️ style_image_recolor not style_img_recolor!

<style> Elements

Definition

<styles>
  <style name="btn"
         bg_color="0x2196f3"
         radius="8"
         pad_all="12"/>
</styles>

Application

<lv_button>
  <style name="btn"/>
  <style name="btn_pressed"
         selector="pressed"/>
</lv_button>
Use same names as style_* but drop prefix!

States & Binding

State Attributes

checked checkboxes, toggle buttons
focused keyboard/input focus
disabled grayed out, non-interactive
pressed button press state

Data Binding

bind_checked to int subject
bind_flag_if_eq conditional flag
bind_state_if_eq conditional state
Syntax: "subject flag value"
